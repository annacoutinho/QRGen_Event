# Mobile App

Aplicativo mobile desenvolvido com **React Native**, **Expo** e **TypeScript**.

## Tecnologias
- React Native + Expo  
- Expo Router (navegação)  
- NativeWind (estilização)  
- Zustand (estado global)  
- Moti (animações)  
- Axios (consumo de API)  
- AsyncStorage (persistência local)  

## Como rodar
```bash
npm install
npm start
